# Agent 3Bd V2 - نظام التصميم

## الفلسفة التصميمية

### المفهوم المرئي
تصميم سيبرني حديث يجمع بين العصرية والوظائفية، مع التركيز على توفير تجربة مستخدم غامرة وحديثة. يعتمد على ألوان داكنة مع لمسات ذهبية لإضفاء طابع مستقبلي ومهني.

### المبادئ التصميمية
- **الوضوح**: واجهة نظيفة وسهلة الفهم
- **الكفاءة**: الوصول السريع للوظائف الأساسية
- **العمق**: استخدام الظلال والتدرجات لإضفاء الطابع الثلاثي الأبعاد
- **التناسق**: توحيد العناصر عبر جميع الشاشات

## الرسوم المتحركة

### المدة
- **الانتقالات السريعة**: 150ms
- **الانتقالات العادية**: 250ms
- **الانتقالات البطيئة**: 300ms

### أنواع الانتقالات
- **ease-out**: للعناصر التي تظهر
- **ease-in**: للعناصر التي تختفي
- **ease-in-out**: للعناصر المتحركة

### الرسوم المتحركة الرئيسية

#### ظهور البطاقات
```css
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

#### نبض الحالة
```css
@keyframes pulse {
  0%, 100% {
    opacity: 1;
  }
  50% {
    opacity: 0.5;
  }
}
```

#### تألق الذهبي
```css
@keyframes shimmer {
  0% {
    background-position: -200% 0;
  }
  100% {
    background-position: 200% 0;
  }
}
```

## التصميم المتجاوب

### الأحجام المعيارية
- **العرض الأقصى للمحتوى**: 400px
- **العرض الأدنى للشاشة**: 320px
- **العرض الأقصى للشاشة**: 768px (للأجهزة اللوحية)

### التصميم للاستخدام بيد واحدة
- **منطقة الإبهام**: جميع العناصر التفاعلية في الجزء السفلي من الشاشة
- **الحد الأدنى للمسافة**: 44px بين العناصر التفاعلية
- **الوصول السهل**: الأزرار الرئيسية في أسفل الشاشة

## القواعد والمعايير

### قواعد التصميم
1. **التباعد**: استخدام مضاعفات 8px للتباعد
2. **الحواف**: استخدام border-radius بقيمة 8px, 12px, أو 16px
3. **الظلال**: إضافة عمق باستخدام الظلال المناسبة
4. **الألوان**: استخدام الألوان المحددة فقط
5. **الخطوط**: الالتزام بنظام الطباعة المحدد

### معايير إمكانية الوصول
- **التباين**: نسبة تباين لا تقل عن 4.5:1
- **حجم اللمس**: لا يقل عن 44x44px
- **النص البديل**: لجميع الصور والأيقونات
- **التسميات**: لجميع الحقول والعناصر التفاعلية

### معايير الأداء
- **وقت التحميل**: أقل من 3 ثوانٍ
- **الرسوم المتحركة**: 60fps
- **الاستجابة**: أقل من 100ms للتفاعلات
